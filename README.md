## 🧭 เกี่ยวกับโปรเจกต์ (About The Project)

**GAME-AI-ROUTINE-ENG** คือสื่อการเรียนรู้ในรูปแบบเกมสไตล์ **Sci-Fi / Cyberpunk Arcade** ที่สร้างขึ้นเพื่อการศึกษาและเรียนรู้ระบบปัญญาประดิษฐ์ (Artificial Intelligence) และเทคโนโลยีความจริงเสริม (Augmented Reality) ช่วยให้ผู้เรียนทั้งเด็กและผู้ใหญ่สามารถทำความเข้าใจลอจิก พฤติกรรม (Routine) และการทำงานของ AI ได้อย่างสนุกสนานและเห็นภาพชัดเจน

> 🚀 **ภารกิจของเรา:** เปลี่ยนทฤษฎี AI ที่ซับซ้อน ให้กลายเป็นเรื่องสนุกผ่านการลงมือทำและโค้ดดิ้ง!

---

## 🌟 ฟีเจอร์เด่น (Key Features)

*   **Cyberpunk UI/UX:** ส่วนติดต่อผู้ใช้งานที่ล้ำสมัย สวยงาม มีมิติ เรืองแสงสไตล์เกมอาร์เคด
*   **AI Routine Visualizer:** ระบบจำลองและแสดงผลการคิดลอจิกของ AI แบบเรียลไทม์
*   **Interactive Learning:** เรียนรู้ผ่านการเล่นและการปรับแต่งค่าคำสั่ง (Parameters) 
*   **Responsive Design:** รองรับการแสดงผลทุกหน้าจออย่างสมบูรณ์แบบ ทั้งบน PC และ Mobile

---

## 🛠️ เทคโนโลยีที่ใช้ (Tech Stack)

*   **Front-End:** HTML5, Tailwind CSS (พร้อมเทคนิค Glassmorphism และ Neon Glow Effects)
*   **Fonts:** 
    *   `Orbitron` - สำหรับข้อความภาษาอังกฤษและธีม Sci-Fi
    *   `Kanit` - สำหรับข้อความภาษาไทยที่อ่านง่ายและโมเดิร์น

## 🚀 เริ่มต้นใช้งาน (Getting Started)

### 1. การติดตั้ง (Setup)
ติดตั้ง Tailwind CSS ผ่าน CDN หรือดาวน์โหลดโปรเจกต์นี้ลงบนเครื่องของคุณ:
```bash
git clone [https://github.com/your-username/GAME-AI-ROUTINE-ENG.git](https://github.com/your-username/GAME-AI-ROUTINE-ENG.git)
GAME-AI-ROUTINE-ENG/
├── assets/          # รูปภาพ ไอคอน และเอฟเฟกต์เสียง
├── css/             # ไฟล์สไตล์สตรีทนีออน (Tailwind Custom)
├── js/              # ลอจิกควบคุมเกมและปัญญาประดิษฐ์
└── index.html       # หน้าจอหลักของโปรเจกต์ (มี Header Component)
